DefinitelyBound

The repository for high quality Xamarin bindings, distributed as NuGet packages.

Usage

Installation is done via the NuGet package manager:

PM> Install-Package $popularPackageName.DefinitelyBound

For example, to install the popular MBProgressHUD into your Xamarin iOS project:

PM> Install-Package MBProgressHUD.DefinitelyBound
